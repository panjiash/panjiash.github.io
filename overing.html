Data Produk

- stok produk + cari produk (kalkulasi otomatis dari produk masuk dan keluar)
- history pengeluaran produk
- history pemasukan produk
- grouping produk per category

Data Pelanggan

- Tambah data pelanggan
- Ubah data pelanggan
- Hapus data pelanggan
- Lihat / cari data pelanggan

Data Karyawan

- Tambah data karyawan
- Ubah data karyawan
- Hapus data karyawan
- Lihat / cari data karyawan

Pembelian Produk

- Tambah data pembelian
- Ubah data pembelian (jika belum di billing)
- Lihat / cari data pembelian
- Otomatis akan menambahkan stok pembelian

Penjualan Produk

- Tambah data penjualan
- Ubah data penjualan (jika belum di billing)
- Lihat / cari data penjualan
- Otomatis akan mengurangi stok penjualan
- Aplikasi hanya mencatat metode pembayaran saja (tidak untuk transaksi material seperti integrasi aplikasi dengan payment gateway)

Laporan

Pembelian Produk

- Rekap pembelian produk per period (yang di beli dari distributor)
- Data yang di provide dalam bentuk grafik
- Data bisa di tarik sesuai periode
- Data bisa di export ke excell
- Data bisa di print dan di simpan sebagai PDF

Penjualan Produk

- Rekap penjualan produk per period (yang di jual ke customer)
- Data yang di provide dalam bentuk grafik
- Data bisa di tarik sesuai periode
- Data bisa di export ke excell
- Data bisa di print dan di simpan sebagai PDF
